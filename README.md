# WeatherToday
iOS 앱 프로그래밍_간단 날씨 앱(연습용)

## 구현 완료 모습

- MVC Design Pattern 적용
- Hard-Coding 하는 부분을 최소화
- JSON data decoding 을 위한 Codable 프로토콜 채택 및 JSONDecoder() 클래스 사용
- UITableView 구현의 기본과 Custom Cell 구현 연습
